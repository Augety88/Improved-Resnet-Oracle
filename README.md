# 甲骨文识别模型：基于改进ResNet18的古文字识别系统

## 项目介绍

本项目实现了一个基于改进ResNet18的甲骨文识别系统，通过引入CBAM注意力模块和边缘增强技术，提高了古文字识别的准确率。项目包含完整的训练、评估和推理功能，并提供了GUI界面方便用户使用。

### 主要特性

- **改进的ResNet18模型**：针对甲骨文特点进行了优化
- **CBAM注意力模块**：增强模型对古文字笔画特征的关注
- **边缘增强技术**：使用多个3×3卷积替代传统7×7卷积，保留更多细节信息
- **完整的训练流程**：支持数据加载、模型训练、评估和模型保存
- **GUI界面**：提供直观的用户界面进行图像识别
- **多指标评估**：支持准确率、F1分数、AUC和Top5准确率等评估指标

## 模型训练

运行训练脚本开始训练模型：

```bash
python train.py
```

训练过程中会自动进行消融实验，比较不同配置的模型性能。训练结果会保存到 `ablation_results.csv` 文件中，模型权重会保存到 `saved_models/` 目录。

## 模型推理

使用GUI界面进行模型推理：

```bash
python infer_oracle_gui.py
```

1. 选择训练好的模型权重文件（.pth）
2. 选择要识别的图像或文件夹
3. 点击"加载模型"按钮
4. 点击"开始识别"按钮查看识别结果

## 模型架构

### OracleResNet18

- **输入层**：3通道RGB图像，大小为224×224
- **边缘增强模块**：3个3×3卷积层替代传统7×7卷积
- **残差块**：4个残差层，其中第3层包含CBAM注意力模块
- **输出层**：包含Dropout层以防止过拟合

### CBAM注意力模块

- **通道注意力**：通过全局平均池化和最大池化获取通道特征
- **空间注意力**：通过通道维度的平均和最大池化获取空间特征

## 📖 项目介绍

本项目是一个基于深度学习的甲骨文智能识别系统，旨在解决古文字识别难度大、准确率低的痛点。通过改进ResNet18网络架构，结合CBAM注意力机制和边缘增强技术，显著提升了对甲骨文手写体和拓片图像的识别性能。

### 🎯 核心价值

- **高精度识别**：针对甲骨文笔画复杂、形态多变的特点进行优化
- **用户友好**：提供图形化界面，无需编程基础即可使用
- **科研支持**：完整的消融实验框架，验证各改进模块的有效性
- **开源开放**：代码完全开源，便于学术研究和二次开发

### ✨ 主要特性

- 🔥 **改进的ResNet18骨干网络**：专为古文字识别优化的深度学习模型
- 🎯 **CBAM双重注意力机制**：通道+空间注意力，精准定位关键笔画特征
- 🔍 **边缘增强技术**：多尺度3×3卷积替代7×7卷积，保留古文字细节信息
- 📊 **多维度评估体系**：准确率、F1分数、AUC、Top5准确率等全面指标
- 💻 **双模式推理**：GUI图形界面 + 批量文件夹识别，灵活便捷
- 🧪 **自动化消融实验**：自动对比不同配置的性能差异

## 🚀 快速开始

### 1️⃣ 安装步骤

```bash
# 克隆项目
git clone https://github.com/Augety88/Improved-Resnet-Oracle.git
cd Improved-Resnet-Oracle

# 安装依赖
pip install -r requirements.txt
```

### 2️⃣ 准备数据集

将甲骨文数据集按以下结构组织：

```
data/甲骨文手写体+甲骨文拓片/
├── train/                  # 训练集
│   ├── 甲类字符/           # 每个类别一个文件夹
│   ├── 乙类字符/
│   └── ...
└── test/                   # 测试集
    ├── 甲类字符/
    ├── 乙类字符/
    └── ...
```

**注意**：每个文件夹名即为类别标签，文件夹内放置对应类别的图像文件（支持.jpg、.png等格式）。

### 3️⃣ 训练模型

```bash
python train.py
```

训练过程会自动：
- ✅ 加载数据集并进行数据增强
- ✅ 执行4组消融实验对比
- ✅ 保存最佳模型权重到 `saved_models/`
- ✅ 生成评估报告 `ablation_results.csv`

**训练时长参考**：根据数据集大小和硬件配置，通常需要数小时至数十小时。

### 4️⃣ 使用GUI进行识别

```bash
python infer_oracle_gui.py
```

**操作步骤**：
1. 点击"选择模型"按钮，加载 `.pth` 权重文件
2. 点击"选择图片"识别单张图像，或"选择文件夹"批量识别
3. 查看识别结果和置信度
4. 支持导出识别结果

## 🏗️ 技术架构

### 模型改进详解

#### 1. 边缘增强模块
```
传统ResNet18:  7×7 Conv → BatchNorm → ReLU
改进方案:      3×3 Conv → 3×3 Conv → 3×3 Conv → BatchNorm → ReLU
```
**优势**：更小的卷积核能够捕捉更精细的边缘特征，适合甲骨文复杂的笔画结构。

#### 2. CBAM注意力机制
在第3个残差层嵌入CBAM模块：
- **通道注意力**：学习哪些特征通道更重要
- **空间注意力**：学习图像中哪些空间位置更关键

**效果**：使模型聚焦于古文字的关键笔画区域，抑制背景噪声。

#### 3. 网络整体架构
```
输入 (224×224×3)
    ↓
边缘增强卷积块
    ↓
残差层1 (64 channels)
    ↓
残差层2 (128 channels)
    ↓
残差层3 + CBAM (256 channels)  ← 注意力机制
    ↓
残差层4 (512 channels)
    ↓
全局平均池化
    ↓
Dropout + 全连接层
    ↓
输出 (N个类别)
```
